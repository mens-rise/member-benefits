匂い対策

 

 

 

 

 

 

前にYouTubeで「【やってないとヤバい】最速でいい匂いになる方法」って動画出した時、20万回近く再生されたんですよ。それだけ匂いに悩んでる男が多いってこと。

 

 

匂いの不安があるってことは、まだ対策が足りてないだけ。正しいアイテムで正しく対策すれば、その不安はゼロにできる。

 

 

匂いって、自分では気づきにくいのがやっかいなんですよ。自分の匂いに鼻が慣れてるから。でも相手は一瞬で気づく。それが体臭でも、足の匂いでも、口臭でも。

 

 

だからここは「気にしすぎ」くらいがちょうどいい。

 

 

匂い対策には3つのエリアがある。体臭・足・口。この3つを全部潰す。

 

 

体臭対策

 

 

 

 

 

 

まず、体臭の原因を知っておいてほしい。

 

 

実は汗そのものはほぼ無臭で、汗が皮膚の雑菌と混ざって時間が経つと臭くなる。

 

 

だから「汗を抑える」か「汗をすぐ乾かす」か「雑菌を繁殖させない」のどれかをやれば、体臭はかなり防げる。

 

 

→ ユニクロ エアリズムインナー（肌色シームレス）

 

 

 

 

これはTシャツの下に着るインナー。汗を吸収してすぐ乾くから、雑菌が繁殖しにくい。

 

 

ポイントは肌色のシームレスタイプを選ぶこと。白Tシャツの下に着ても透けないし、首元や袖から見えない。

 

 

夏は特にマスト。Tシャツの脇に汗ジミが出るのも防いでくれる。1,500円くらいで買える体臭対策としてはコスパが異常。

 

 

俺は最低3枚ローテで回してる。洗い替え含めて。

 

 

→ オールドスパイス ピュアスポーツ

 

 

 

 

制汗剤。脇に直接塗るタイプ。

 

 

日本のスプレータイプの制汗剤より、海外のスティックタイプの方が持続力が圧倒的に強いんですよ。朝塗ったら夜まで持つ。

 

 

しかも「ピュアスポーツ」は香りが控えめやから、香水の邪魔にならない。制汗剤の匂いと香水が混ざって変な匂いになるパターン、よくあるんやけど、これならそれがない。

 

 

朝、シャワー浴びた後に脇に2〜3回スライドさせるだけ。

 

 

足の匂い対策

 

 

 

 

 

 

足の匂いって、自分では気づきにくいけど、靴を脱ぐ場面（居酒屋の座敷とか）で一発でバレる。

 

 

デートで座敷の店を避けてるなら、それは対策が足りてないだけ。

 

 

→ グランズレメディ

 

 

 

 

靴の中に粉を入れるだけ。これ、マジで魔法みたいに効く。

 

 

1週間くらい毎朝靴に振りかけて履くだけで靴の匂いがほぼゼロになる。

 

 

仕組みとしては、粉が靴の中の雑菌を抑制してくれる。俺は新しい靴を買ったら、まずこれを入れるところから始める。

 

 

→ LUSH 素足のTブレイク

 

 

 

 

足そのものをケアするフットソープ。グランズレメディが「靴」の対策なら、これは「足」の対策。

 

 

週2〜3回、お風呂でこれで足を洗うだけ。足の裏の角質や雑菌をケアしてくれて、匂いの元を断ってくれる。

 

 

→ デオナチュレ 足指サラサラクリーム

 

 

 

 

出かける前に足の指の間に塗るクリーム。汗を抑えてサラサラにしてくれる。

 

 

グランズレメディ（靴）× LUSH 素足のTブレイク（お風呂）× デオナチュレ（出かける前）。
この3つを組み合わせたら、足の匂い問題はほぼ完全に解決する。

口の匂い対策

 

 

 

 

 

 

これ、一番盲点になってるやつ。体臭と足は気にしてても、口臭を放置してる人がめちゃくちゃ多い。

 

 

相手との距離が近くなった瞬間にバレるのが口臭で、しかも指摘してくれる人がほとんどいないから、自分では気づかんまま過ごしてしまう。

 

 

口臭の原因の大半は舌の汚れなんですよ。歯を磨いても口臭が消えん人は、まず舌を見てほしい。白くなってたら、それが原因。

 

 

→ NONIO 舌クリーナー＋クリーニングジェル

 

 

 

 

舌専用のクリーナー。歯ブラシで舌を磨いてる人もおるけど、舌の表面は繊細やから専用のやつを使った方がいい。

 

 

朝の歯磨きの後に、ジェルをクリーナーに付けて舌をやさしく3〜4回撫でるだけ。10秒で終わる。

 

 

→ コンクール

 

 

 

 

マウスウォッシュ。歯磨きの後にこれで口をゆすぐ。

 

 

市販のマウスウォッシュは刺激が強くて使いにくいやつも多いけど、コンクールは歯医者でも推奨されてるやつで、刺激が少ないのに殺菌力がしっかりある。

 

 

歯磨き → 舌クリーナー → コンクール。
この3ステップを毎朝やれば、口臭で困ることはなくなる。

あと、歯間ケアもやった方がいい。歯ブラシだけだと歯と歯の間の汚れは取れないんですよ。

 

 

  • フロアフロス — 歯間の汚れをゴッソリ取ってくれる。使い始めたら「今までどんだけ汚れ残ってたんや」ってなる
  • Oral-B フロスピック — フロスが苦手ならこっち。ピック付きで使いやすい

 

 

匂い対策だけで7アイテム紹介した。「多いな」って思うかもしれんけど、全部合わせても5,000円くらいで揃う。

 

 

月5,000円で「この人いい匂いする」って思われる投資やと考えたら、安すぎると思う。

 

 

匂いの土台ができたら、次はスキンケア。肌が綺麗なだけで、清潔感の印象がまるで変わるんですよ。

 

 

 

 

スキンケア

 

 

 

 

 

 

「スキンケアとか女がやるもんやろ」って思ってる人、まだおるかもしれんけど2026年の今、肌が汚い男は「手入れしてない人」って見られる。

 

 

女性がメイクするのと同じ感覚で、男もスキンケアは最低限やっとくべき時代になってる。

 

 

しかも男のスキンケアって、そんなに難しくないんですよ。高い化粧品を何個も使う必要はない。最低限のことを毎日やるだけで、肌の印象がかなり変わる。

 

 

まず全員やってほしい：日焼け止め

香水

香水のつけ方（これ間違えてる人むちゃくちゃ多い）

 

 

 

 

 

 

ステップやること
1手首に1プッシュ → ぽんぽんと軽く馴染ませる
2もう1プッシュ手首に → 首にぽんぽんと移す

 

 

絶対にやったらあかんこと：

 

 

  • 擦らない。手首同士をゴシゴシ擦る人がおるけど、摩擦熱で香りの成分が飛んで、本来の匂いが変わってしまう。ぽんぽん、と置くように馴染ませる
  • 5プッシュとかつけすぎない。俺、昔5プッシュくらいつけて友達に「臭いわ」って言われた。2プッシュで十分。「ちょっとだけ香る」くらいがちょうどいい

発汗作用があるから、めちゃくちゃ汗が出る。老廃物が出る感覚がある。ラベンダーの香りでリラックスもできるから、寝つきが良くなる。

 

 

→ CBDバスソルト

 

 

 

 

こっちはCBD成分入りのバスソルト。リラックス効果がさらに強い。

 

 

俺は夜のルーティンとして、エプソムソルトかCBDバスソルトのどっちかを入れてお風呂に入るようにしてる。

 

 

→ ストレッチポール＋ヨガマット

 

 

 

 

寝る前にストレッチポールの上に仰向けに乗って、5分間ゴロゴロするだけ。背中がバキバキに凝ってる人はこれやるだけで体がめちゃくちゃ軽くなる。

 

 

筋トレしてる人は特に、ストレッチやリカバリーをサボりがちなんやけど身体のケアを怠ると怪我するし、猫背になるし、見た目にも出る。

 

 

5分だけでいいからやってほしい。

 

 

ここまでが「外側」のケア。ここから先は「中身」を変えていくパート。サプリメントの話。

 

 

 

 

サプリメント

 

 

 

 

 

 

正直に言うと、サプリメントは即効性がない。飲んだ次の日に何か変わるわけじゃない。

 

 

でもな。3ヶ月続けたら、明らかに身体の調子が違ってくる。

 

 

肌のツヤ、朝の目覚め、筋トレのパフォーマンス、髪質。全部じわじわ変わってくる。

 

 

俺がサプリを続けてる理由は、「何もしなかったら老化にだけ進んでいく」ことを知ってるから。サプリはその進行を遅らせてくれる、もしくは今の状態をキープしてくれるもの。

 

 

ただし、サプリはあくまで食事の補助。ジャンクフードばっか食って、サプリだけ飲んでも意味ない。まず食事を整えた上で、足りない部分をサプリで補うっていう考え方が大事。

 

 

プロテイン

 

 

 

 

 

 

→ アンビーク プロテイン ヨーグルト味

 

 

 

 

俺のメインプロテイン。ヨーグルト味がマジでうまい。プロテイン特有の「粉っぽさ」がほとんどない。

 

 

朝のスムージーに入れたり、オートミールに混ぜたりして飲んでる。

 

 

筋トレしてない人でも、タンパク質は意識して摂った方がいい。肌も髪も爪もタンパク質でできてるから、不足すると見た目に出る。

 

 

1日1杯、朝ご飯と一緒に飲むだけでいい。

 

 

トレーニング系サプリ

 

 

 

 

 

 

筋トレやってる人向け。やってない人はスキップしてOK。

 

 

→ VITAS シトルリン＆アルギニン

 

 

 

 

パンプアップ系のサプリ。トレーニング前に飲むと血流が良くなって筋肉がパンパンに張る。見た目のモチベーションにもなるし、トレーニングの質も上がる。

 

 

→ クレアチン

 

 

 

 

筋力アップのサプリ。トレーニングのパフォーマンスが上がる。1日5gを水に溶かして飲むだけ。味はほぼない。

 

 

健康・美容系サプリ

 

 

 

 

 

 

→ VITAS ビタパワー マルチビタミン

 

 

 

 

美容と健康の土台になるマルチビタミン。ビタミンB群、C、E、亜鉛とか全部入ってる。

 

 

肌荒れしやすい人、疲れやすい人は、まずこれから始めてみてほしい。

 

 

→ ネイチャーカン アシュワガンダ（KSM-66）

 

 

 

 

睡眠の質を改善するサプリ。

 

 

俺がこれを飲み始めたのは、夜なかなか眠れない時期があったから。飲み始めて4〜8週くらいで、入眠が明らかに早くなったんですよ。体感で10分くらい。

 

 

睡眠の質が上がると、翌日の肌のツヤも、トレーニングのパフォーマンスも、集中力も全部上がる。睡眠は男磨きの最強のブースター。

 

 

→ ネイチャーカン CBDオイル

 

 

 

 

リラックス用。寝る30分前に舌の下に数滴垂らして飲む。

 

 

アシュワガンダが「入眠を早くする」なら、CBDオイルは「リラックス状態を作る」イメージ。寝る前のルーティンに組み込むと、睡眠の質がさらに上がる。

 

 

美白・美肌

 

 

 

 

 

 

→ 美容内服薬（グルタチオン、Lシステイン、シナール）

 

 

これは皮膚科で処方してもらうやつやから、市販では買えない。

 

 

肌を内側から白く、綺麗にしてくれる。外側のスキンケアだけでは限界がある部分を、中からケアするイメージ。

 

 

興味がある人は、皮膚科に行って「美容内服薬を試したいんですけど」って相談してみてほしい。月3,000〜5,000円くらいで始められるところが多い。
